CSS Grid Layout, is a two-dimensional grid-based layout system with rows and columns, making it easier to design web pages without having to use floats and positioning. Like tables, grid layout allow us to align elements into columns and rows. The CSS Flexbox offers a one-dimensional layout. It is helpful in allocating and aligning the space among items in a container (made of grids). It works with all kinds of display devices and screen sizes.

Bootstrap is the most popular HTML, CSS, and JavaScript framework for building responsive,mobile-first projects on the web. Whereas Tailwind CSS is a utility-first CSS framework for creating custom user interfaces quickly.

Every element that can be displayed on a web page is comprised of one or more rectangular boxes.CSS Box Model typically describes how these rectangular boxes are laid out on a web page.Every box has a content area as well as optional padding, border, and margin portions.

HTML semantics refers to the tags in an HTML page that provide meaning rather than just display. It clarifies HTML by better describing the various sections and layout of web pages.
